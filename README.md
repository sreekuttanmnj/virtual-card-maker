# Virtual Card Maker

Static marketing site for the Virtual Card Maker product — issue virtual Visa cards for payroll, vendors, subscriptions, and team expenses, with spend limits, merchant locks, and per-transaction tracking.

## Structure

- `index.html` — main landing page
- `about.html`, `contact.html` — informational pages
- `privacy-policy.html`, `terms-and-conditions.html`, `disclaimer.html` — legal pages
- `legal-style.css` — shared stylesheet for the legal pages

## Local preview

Open `index.html` directly in a browser, or serve the folder:

```sh
python -m http.server 8000
```

Then visit http://localhost:8000.
